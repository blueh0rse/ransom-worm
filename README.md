First clone the repository:

````bash
git clone https://github.com/blueh0rse/ransom-worm
````

Move to working directory:

````bash
cd ransom-worm
````

Install depedencies:

````bash
python3 -m pip install -r requirements.txt
````

````bash
python3 main.py
````
